How It Works

From local lux readings to estimated UV risk

The VEML6030 measures lux, not UV. Weather UV and daylight conditions provide external context for estimating local exposure risk.

System Flow

VEML6030 lux reading
ESP32 processing
Estimated UV calculation
OLED + LED + buzzer output
Bluetooth data transmission
Dashboard update
Weather comparison
Recommendations

Local Sensor

The VEML6030 provides real local ambient brightness in lux, which helps detect shading, obstruction, brightness changes, and lighting conditions.

Weather Context

The Flask backend can use weather UV, cloud cover, sunrise, sunset, and weather condition data to understand the expected environmental risk.

Alerts

The ESP32 OLED, LEDs, and buzzer provide local feedback while the dashboard stores readings and displays trends.
